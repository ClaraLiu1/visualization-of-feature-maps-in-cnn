# Visualization-of-feature-maps-in-cnn
    Visualize feature maps in convolutional neural networks, based on Tensorflow and Matplotlib.
## Library versions
  * python 3.5.2;
  * tensorflow 1.4.1;
  * numpy 1.14.0 ；
  * matplotlib 1.5.3;
